# Explorador de archivos — Demo

Este es un **Markdown de ejemplo** para probar la _vista previa_.

## Funciones
- Vista previa de imágenes, código y Markdown
- Navegación por teclado
- Búsqueda instantánea

> Suelta este proyecto en tu htdocs y navega cualquier carpeta.

const saludo = (nombre) => `Hola, ${nombre}!`;
